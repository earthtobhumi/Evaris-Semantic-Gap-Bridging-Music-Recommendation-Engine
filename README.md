# Evaris

A semantic gap-bridging music recommendation engine that recommends songs based on emotional similarity rather than listening history.

## Problem

Traditional recommendation systems rely on collaborative filtering and often miss the emotional connection between songs. Evaris bridges this gap using multilingual NLP embeddings, human-in-the-loop sentiment annotations, Reddit crowd sentiment, and audio DSP features.

## Features

- Multilingual recommendation across 6 languages
- Human-in-the-loop sentiment modeling
- Reddit crowd sentiment enrichment
- Audio DSP analysis using Librosa
- Streamlit interface
- Explainable recommendations

## Tech Stack

Python, SQLite, Pandas, NumPy, Sentence Transformers, Librosa, Streamlit
